# Panda Detector tutorial moved

The detector tutorial is now part of {doc}`Explore Panda <explore_panda>`. The
new page starts with one PILArNet-M-mini test event, runs the released particle
and interaction checkpoints, shows the raw and postprocessed output contracts,
and includes genuine prediction-versus-truth figures.

- {doc}`Explore Panda <explore_panda>` for the base encoder, semantic
  predictions, particles, and interactions.
- {doc}`Fine-tuning <../workflows/fine_tune>` to turn the walkthrough into a
  controlled research experiment.
